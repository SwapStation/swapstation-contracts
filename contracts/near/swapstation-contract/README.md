## SwapStation NEAR Smart Contract

#### Install dependencies
`cd contracts/near/swapstation-contract`
`yarn`

#### Build
`yarn build`

#### Login NEAR CLI
`yarn login-near`

#### Deploy to DEV
`yarn deploy:dev`

#### Deploy to TESTNET (contract.swapstationio.testnet)
`yarn deploy:testnet`

#### Deploy to MAINNET (contract.swapstationio)
`yarn deploy:mainnet`
